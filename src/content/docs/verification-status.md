# Verification Status

| Status | Meaning |
|---|---|
| Verified | Backed by a listed source. |
| Inferred | Name or concept is public, but classification or behavior is inferred. |
| Needs verification | Placeholder or incomplete information. |

Use only these three statuses in data files.
