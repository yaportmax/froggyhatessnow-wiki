# Beginner Guide

Start from the verified public loop: leave home, dig through snow, collect resources, return value home, and use progression to grow stronger.

## Confirmed From Public Sources

- You can survive by finding the escape door or defeating the boss.
- Keys matter for the escape door and can also be spent on treasure chests.
- Warmth matters; staying too long in the cold can freeze you.
- Peaceful Mode is described as monster-free.
