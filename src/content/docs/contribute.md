# Contribute

Contributions should include a source and one of the allowed verification statuses: Verified, Inferred, or Needs verification.

Do not add proprietary assets, binaries, source code, decompiled content, DRM-bypassed material, or long raw text dumps from the game files.

Useful contributions include screenshots of public pages, short gameplay notes, safe metadata findings, and corrections to inferred categories.
